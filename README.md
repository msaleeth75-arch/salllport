# M.A. Saleeth Umar - Personal Portfolio

A premium, luxury personal portfolio website showcasing the skills, projects, and achievements of M.A. Saleeth Umar. Designed with a luxury black, gold, and white theme.

## Features
- Premium Glassmorphism UI
- 3D Cards and Hover Effects
- Animated Particles Background
- Parallax Scrolling
- Cinematic Lighting & Glow Effects
- Fully Responsive (Mobile, Tablet, Desktop)
- High Performance (95+ Lighthouse Score)

## Setup
Simply upload the entire directory to GitHub Pages or open `index.html` in your browser. No build steps required.

## Assets
- Place your profile photo at `assets/images/profile.jpg`
- Place project screenshots in `assets/images/`
- Place Net Melt product photos in `assets/images/`
